Для запуска демки необходимо: 

запустить скрипт =============> demo.py
открыть браузер на странице ==> http://127.0.0.1:5000/
добавить отзыв на русском в текстовое поле формы
нажать кнопку     ============> определить тип отзыва

======================================================================

В случае возникновения ошибок, связанных с несовместимостью библиотек:

удалите файл =================> trained_classifier.pickle
запустите скрипт =============> classifier.py
попробуйте заново запустить демку
